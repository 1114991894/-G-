## 如何运行

### 1. 后端API服务
```bash
cd api
npm install
cp .env.example .env
# 编辑 .env 配置数据库连接
npm run dev
# 访问: http://localhost:3000
```

### 2. 客户端前端
```bash
cd client/web
npm install
npm run dev
# 访问: http://localhost:5173
```

### 3. 服务端前端
```bash
cd server/web
npm install
npm run dev
# 访问: http://localhost:5174
```
